## 🚀 How to Run the Project

1. **Navigate to the project directory:**
   ```bash
   cd E:\zesa_ai_assistent
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000`
